# DTMSim-IoT
DTMSim-IoT is an open-source simulator for Distributed Trust Management Models.

For instance, following two distributed trust management models are implemented:
1. Dynamic Distributed Trust Management Model
2. Static Distributed Trust Management Model

*Note: References to the aforementioned models will be provided on request. 

However, the other distributed trust management models can also be implemented using this source code. If anyone wants to contribute, please let me know.

*Note: In case of any inconvinience, please don't hesitate to contact me.
