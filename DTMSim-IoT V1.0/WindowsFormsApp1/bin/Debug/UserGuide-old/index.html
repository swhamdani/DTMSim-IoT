DDTMS User Guide

Dynamic Distributed Trust Management System Simulation Tool

Simulator Information

Menu:

This is menu bar consists of File and Help menus. 

Network Area:

This shows the IoT network consisting of various nodes.

Network Logs:

It shows log of every transaction during different network operations.

Network Operations:

This panel includes various network operations e.g. Create Network, Service Request etc.

Click on "File"

File Menu consist of Preferences, Logs and Exit Options.

Click on "Preferences"

On Preferences screen, 

User can enter delay time of each transaction, in milliseconds.

User can enter total number of transactions per node. After entering values, click on Save button. It saves these values as default preferences. 

Click on "Dynamic"

Select Distributed Network type i.e. Dynamic or Static. 

Enter No. of "Nodes"

Type in: 20 for total network nodes.

Click on "Create Network"

After entering the total number of nodes, click on this button. It will create nodes of the network.

Click on "Get Initial Info"

In this network operation, neighbor discovery is done, in which each node sends packets to its neighbor nodes. Trust table of each node created which contains information about neighbor nodes.

Service Request

 By clicking this button, service request operation started. Each node in the network sends random request to its neighbor node from its trust table. Every request initiated after delay time.

Service Request...

Node sending random service request to its neighbor node from its trust table. Every request initiated after delay time. 

Click on "Service Evaluation"

Once service request operation done, then service evaluation operation can be initiated. It evaluates the node's reward score and penalty score based on the service provision or service rejection respectively.

Click on "Compute Trust"

Finally, trust of each node is computed based on Reward and Punishment values.

Malicious Node

On trust computation, Malicious node is marked red circle and added to nodes BlackList.

Click on "Logs"

To view network logs and Trust Table of each node in the network.

Click on "Print Trust Table"

This screen shows Trust Table and Network logs of Dynamic / Static models.

Trust Table

It displays Trust Table abstract values of each node i.e. neighbor list, services offered and trust value.

Node 2 has 2 neighbors i.e. node 12 and node 20.

Click on "Get Network Logs"

It displays Trust Table detailed information consisting of each node's every single transaction and trust value.
