Click on "File"

File Menu consist of

Preferences, Logs and Exit

Options.

Click on "Preferences"

On Preferences screen,

User can enter delay time of

each transaction, in

milliseconds.

Type in: 5

Enter Total Nodes

User can tap up-arrow / down-arrow to enter total number of nodes.

Click on "Configure Network"

After entering the total

number of nodes, click on this

button. It will create nodes of

the network.

Click on "Initialize Network"

In this network operation,

neighbor discovery is done, in

which each node sends

packets to its neighbor nodes.

Trust table of each node

created which contains information about neighbor nodes.

Click on "Service Request"

By clicking this starts the service request operation.

Service Request...

By clicking this button,

service request operation

started. Each node in the

network sends random

request to its neighbor node

from its trust table. Every

request initiated after specified delay time.

Service Request...

The Top right button takes and saves screenshot. Progress bar shows the overall progress. The highlighted node 3, requesting services randomly from its neighbour nodes randomly. Log shows current transaction / service request detail including time, id, requester node id, the service requested and provider node id.

Click on "Service Evaluation"

Once service request

operation done, then service

evaluation operation can be

initiated. It evaluates the

node's reward score and

penalty score based on the service provision or service rejection

respectively.

Service Evaluation...

 

Evaluating node(s) Reward OR Punishment values based on provided or rejected service. 

Click on "Compute Trust"

Finally, trust of each node is

computed based on Reward

and Punishment values.

Compute Trust...

Malicious Node: On trust computation, Malicious node is marked red circle (as shown highlighted) and added to BlackList. Blastlist shows Node ID:7 detected as malicious node. Node Info illustrates node serviceslist, trust value and total neighbours.

Click on "Logs"

To view network logs and

Trust Table of each node in

the network.

IoT Network Logs

It shows Trust table and Network logs of Dynamic / Static Network. 

Click on "Print Trust Table"

It displays trust table of all nodes in the network. The highlighted area depicts the trust table of Node: 1, consisting of its neighbours, their services list and trust values.

Click on "Get Network Logs"

It displays Trust Table detailed information consisting of each node's every single transaction and

trust value.

Get Network Logs...

Lists each node complete transaction history. Highlighted one depicts transaction history of Node:1, consisting of each transaction with each neighbour.
